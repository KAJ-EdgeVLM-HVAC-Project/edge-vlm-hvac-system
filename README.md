# 📄 PRD: 엣지 VLM 및 IoT 융합 기반 완전 오프라인 지능형 공조(HVAC) 제어 시스템

## 프로젝트 요약 (Executive Summary)
본 프로젝트는 소형 임베디드 보드(NVIDIA Jetson)의 제한된 리소스 내에서 시각-언어 모델(VLM)을 최적화하여 구동하는 **스탠드얼론(Standalone) 스마트 공조 제어 시스템**을 개발합니다. 외부 클라우드 네트워크 연결 없이 기기 내부에서 실내 상황(재실자 수, 착의량, 활동량)을 자체적으로 인지함으로써 네트워크 지연 및 사생활 침해 문제를 원천 차단합니다. 단순한 온도 센서 의존을 넘어, 재실자의 '열 쾌적성(PMV)'을 선제적으로 예측하여 에너지를 절감하는 산업용 레트로핏(Retrofit) 솔루션을 제안합니다.

---

## 프로젝트 목표 (Project Objectives)
- **오프라인 엣지 VLM 파이프라인 구축:** 카메라 이미지로부터 공간의 복합적인 맥락(Context)을 파악하는 VLM을 엣지 디바이스에 포팅하고, 양자화(Quantization) 및 가속화를 통해 실시간 추론 성능을 확보합니다.
- **하이브리드 자율 제어 시스템 구현:** VLM이 추출한 비정형 공간 상황 데이터(정성적)와 IoT 센서가 측정한 기기 상태 데이터(정량적)를 융합하여, 쾌적성을 극대화하면서도 전력 소모를 최소화하는 지능형 제어 로직을 완성합니다.

## 핵심 차별성 (Key Features)

### ① 완전 오프라인 엣지 AI (Privacy-Preserving Edge Computing)
- **차별점:** 카메라가 촬영한 이미지는 외부 서버로 전송되지 않으며, 임베디드 보드의 RAM 내부에서 1회성 추론에만 사용된 직후 영구 파기됩니다.
- **기대효과:** 네트워크 지연(Latency) 없는 즉각적인 처리가 가능하며, `{"사람 수": 4, "옷차림": "두꺼움"}` 형태의 텍스트 변수만 제어에 활용하므로 보안 구역에서도 사생활 침해 제로(0)를 보장합니다.

### ② 비전 기반 열 쾌적성(PMV) 추정 시스템
- **차별점:** 기존 센서로는 측정 불가능한 재실자의 착의량(Clo, 옷의 두께), 활동량(Met, 움직임의 강도), 더워하는 행동 패턴 등을 VLM이 종합적으로 시각 인지합니다.
- **기대효과:** 온도가 올라간 후 에어컨이 켜지는 기존의 '후행 제어'가 아닌, 사용자가 불쾌감을 느끼기 전 예상 온열감(PMV)을 계산해 대응하는 '선제적 제어'를 통해 에너지를 효율적으로 관리합니다.

### ③ VLM + IoT 하이브리드 아키텍처 (연산 최적화)
- **차별점:** VLM의 무거운 연산력을 에어컨 디스플레이 숫자를 읽는 등의 단순 작업에 낭비하지 않고, 비정형 공간 맥락 인식에만 100% 집중시킵니다.
- **기대효과:** 현재 실내 온도 및 기기 설정값은 가벼운 IoT 통신으로 오차 없이 수집하여 융합함으로써, 임베디드 환경에서의 램(RAM) 점유율을 방어하고 시스템 안정성을 극대화합니다.

## 특화 적용 시나리오 (Application Scenarios)
기존 IoT 센서의 고질적 한계(물리적 반응 지연, 맥락 인지 불가)를 극복하기 위해 아래 3대 특화 환경을 타겟팅합니다.

- **시나리오 A: 다이내믹 피트니스 센터 (구역별 열 쾌적성 맞춤 제어)**
  - **문제점:** 헬스장은 구역별 활동량 차이가 극심하나, 중앙 에어컨은 전체 공간의 평균 온도만 측정합니다.
  - **VLM 솔루션:** 러닝머신 구역의 고강도 활동(High Met) 및 땀을 닦는 행동과, 스트레칭 구역의 정적 활동(Low Met)을 시각적으로 구분하여, 고강도 구역에 국소적으로 풍량을 집중시킵니다.
- **시나리오 B: 상업용 조리 시설 (센서 지연 극복 선제적 제어)**
  - **문제점:** 주방 화구에서 열기가 발생하여 천장 센서에 닿고 에어컨이 반응하기까지 수 분의 지연 시간(Time-lag)이 발생합니다.
  - **VLM 솔루션:** VLM이 화구의 점화, 수증기 발생, 작업자의 분주한 움직임을 즉시 인지하여, 물리적 온도가 상승하기 전에 냉방 및 환기 시스템을 최대 출력으로 사전 가동합니다.
- **시나리오 C: 스마트 오피스 트랜지션 (공간 맥락 인지 에너지 절감)**
  - **문제점:** 사람이 완전히 퇴실한 후에야 센서가 반응하여 빈 방에 불필요한 냉난방 에너지가 소비됩니다.
  - **VLM 솔루션:** 직원들이 외투를 입고 가방을 싸는 '퇴근 준비 맥락'이 포착되면, 잔류 인원이 0이 되기 전 공조기를 미리 종료하여 잔열/냉기로 공간을 유지해 에너지를 획기적으로 절약합니다.

### 하드웨어 (Hardware)
- **선정 기기:** **NVIDIA Jetson Orin Nano Super (8GB)**
- **선정 사유:** VLM 구동을 위한 강력한 Tensor 코어 성능과 8GB의 공유 메모리(UMA)를 제공하며, 한정된 대학생 프로젝트 예산 내에서 모델을 하드웨어 가속(TensorRT)하여 오프라인 구동할 수 있는 최적의 엣지 폼팩터입니다.

### 소프트웨어 (Software)
- **선정 모델:** **Qwen2.5-VL (3B 파라미터)**
- **선정 사유:** 30억 개의 초경량 파라미터로 엣지 보드 탑재가 가능하며, 현존 경량 VLM 중 시각적 디테일 파악 능력과 한국어 지시 기반 프롬프팅 수행 능력이 가장 뛰어납니다. JSON 포맷 파싱을 통한 시스템 연동에 최적화되어 있습니다.

### 데이터 파이프라인 (Data Flow)
```plaintext

       ├─> 현재 공간의 열 쾌적성(PMV) 및 예측 냉난방 부하 계산
       └─> 에어컨/환기팬 제어 신호 송신 (온도 조절, 풍량 집중 등)
```

## 상업화 전략 및 비즈니스 기대효과

- **레거시(Legacy) 인프라의 스마트화 (Retrofit 솔루션):** 수천만 원에 달하는 최신 스마트 에어컨으로 교체할 필요 없이, 기존 구형 공조기기에 엣지 보드를 부착하기만 하면 최신 AI 솔루션으로 탈바꿈하여 프랜차이즈 시장 진입 장벽을 낮춥니다.
- **즉각적인 ROI 및 ESG 경영 부합:** 낭비되는 공조 에너지를 15~20% 절감함으로써 단기간 내 시스템 도입 비용 회수(ROI)가 가능합니다. 이는 상업 빌딩의 탄소 배출 저감이라는 ESG 경영 지표 달성에 직접적으로 기여합니다.
- **공간 데이터 수익화 (Data Monetization):** VLM이 안전하게 추출한 비식별 텍스트 데이터(특정 시간대 고객 밀집도, 체류 시간, 주요 활동 구역)를 활용해 매장 점주에게 '상업 공간 효율성 분석 대시보드'를 SaaS 형태로 추가 제공할 수 있습니다.

## 🚀 16주 개발 마일스톤 (16-Week Roadmap)

### Part 1: 기반 구축 및 VLM 프로토타이핑 (Weeks 1-4)

- **[⏳] 3주차:** PC 기반 개발 환경 구축, VLM(Qwen2.5-VL) 모델 성능 및 리소스 점유율 테스트
- **[🗓] 4주차:** 도메인 특화 VLM 프롬프트 엔지니어링 및 출력 데이터 JSON 정규화(Parsing)

### Part 2: 엣지 디바이스 배포 및 최적화 (Weeks 5-8)
- **[🗓] 5주차:** NVIDIA Jetson Orin Nano 개발 보드 OS/환경 설정 (JetPack-6.0)
- **[🗓] 6주차:** Jetson 보드 환경으로 VLM 모델 포팅 및 오프라인 로컬 추론 테스트
- **[🗓] 7주차:** 모델 경량화(INT4 양자화 적용) 전후 메모리 및 성능 비교
- **[🗓] 8주차:** TensorRT 엔진 변환을 통한 추론 속도(FPS) 극대화 작업

### Part 3: IoT 연동 및 파이프라인 통합 (Weeks 9-12)
